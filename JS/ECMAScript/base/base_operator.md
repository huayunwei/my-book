### 优先级

一元运算符 &gt; 算术运算符 &gt; 比较运算符 &gt; 逻辑运算符 &gt; 三元运算符 &gt; 赋值运算符 &gt; 逗号运算符

可以用圆括号来强行执行运算顺序，括号内的值会被运算，再和外部的进行运算

### 结合性

左结合：从左向右结合，多数运算符

```
w = x + y + z;
w = ((x + y) + z );
```

右结合：从右向左结合，一元运算符、条件运算符和赋值运算符

```
w = x = y = z;
w = (x =(y = z));
```
