## Input

* name：input元素的名称，用于对提交到服务器后的表单数据进行标识

* type：规定input元素的类型

  * chekbox

  * file

  * hidden

  * image

  * password

  * radio

  * reset

  * submit

  * text

  * color

  * date

  * datetime

  * datetime-local

  * email

  * month

  * number

  * range

  * search

  * tel

  * time

  * url

  * week
