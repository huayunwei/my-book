## Input

* name：input元素的名称，用于对提交到服务器后的表单数据进行标识

* type：规定input元素的类型

  * button

  * checkbox

  * file

  * hidden

  * image

  * password：密码框，和text外观一样，但是输入的内容不可见

    * 通过将input的type属性修改为text可以达到密码可见的效果

  * radio

  * reset

  * submit

  * text：单行输入字段，可以输入文本

  ![](/assets/label_input_text.png)

  * color

  * date

  * datetime

  * datetime-local

  * email

  * month

  * number

  * range

  * search

  * tel

  * time

  * url

  * week
