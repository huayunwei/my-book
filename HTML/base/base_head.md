## HEAD下的子元素主要有以下几种

### meta

* charset：声明当前文档所使用的字符编码

```markdown
<meta charset="utf-8"/>
```

* name：搭配content使用，定义一些文档级的元数据

```markdown
//关键词
<meta name="keywords" content="前端 总结">

//描述
<meta name="description" content="前端知识总结">

//作者
<meta name="author" content="">

//版权
<meta name="copyright" content="">

//视口
```
